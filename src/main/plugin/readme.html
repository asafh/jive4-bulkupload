Usage

By default, there is a link to the utility under:

  • Profile Actions (Self)
  • Profile Actions
  • Profile Overview (i.e. User's Stuff)

other locations can be added by either editing plugin.xml or adding a link to the action bulk-upload!inputLocation.jspa (i.e. action bulk-upload method inputLocation)

Users can only use the tool to upload to locations they'd noramlly be able to upload (sinlge) documents to
